# Hash Tables

This project implements hash tables in C, including collision handling using chaining.

## Files

- `hash_tables.h` - Header file with all prototypes
- `0-hash_table_create.c` - Creates a hash table
- `1-djb2.c` - Implements the djb2 hash function
- `2-key_index.c` - Gets the index of a key
- `3-hash_table_set.c` - Adds an element to the hash table
- `4-hash_table_get.c` - Retrieves a value associated with a key
- `5-hash_table_print.c` - Prints a hash table
- `6-hash_table_delete.c` - Deletes a hash table

## Compilation
```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hash_test
```
